Welcome to GAEStarterKit

Try out GAEStarterKit

  • Download the source along with the App Engine SDK
  • Create config.py, using config-sample.py as an example
  • Run dev-appserver.py .
  • Try various login methods, including,
    • Email+password
    • Google login
    • Social login (github, etc). You'll need to put your OAuth credentials in config.py, of course.
  • Take a gander at your user profile page and
    • Try adding multiple email addresss,
    • Set your display name.
    • Change your password.
  • Browse the admin GUI dashboard. Make sure you authenticate with Google as a superuser, and you can:
    • View pages, users, tenants, and emails.
    • See recent activity within application.
  • Create a tenant account and
    • Invite other users.
    • Create multiple tenants and switch between them fluidly.
